# Exploring Windows Defender Events  

Researching:
- What are the event paths?  
- Can we create an event framework for event CRUD operations?
- Can we create events?  
- What are the security rules for each path (are some more locked down than others)?
- Where are these events stored on disk?  

Compile the dll  
```powershell  
csc /target:library /out:payload.dll get-event-paths.cs
```  

Load the Dll method 1 (local)
```powershell  
$currentDir = Get-Location
$bytes = [IO.File]::ReadAllBytes("$currentDir\payload.dll")
$assembly = [System.Reflection.Assembly]::Load($bytes)
$instance = $assembly.CreateInstance("SecurityTools.DefenderHistory")
$instance.GetEventLogPaths()
```
